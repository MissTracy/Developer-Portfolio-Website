# 💼 Personal Developer Portfolio

A moderncresponsive developer portfolio showcasing my skills and some projects as a Full Stack Developer.

---

## ✨ Overview

This portfolio was built to present my experience, projects, and technical skills in a clean and interactive way. It highlights my journey as a developer and provides an easy way to connect with me.

---

## 🌟 Live Demo

👉 [Click here for live demo (netlify)](https://misstracyportfolio.netlify.app)

---

## ⚡ Features

-  Smooth responsive layout (mobile + desktop)
-  Animated typing effect on homepage
-  Scroll reveal animations for sections & project cards
-  Project showcase grid (2x2 layout on large screens)
-  Modern dark UI with accent colors
-  Fully mobile-friendly design
-  Clickable social links (GitHub, LinkedIn, Email)

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox + Grid + Animations)
- JavaScript (DOM manipulation + Intersection Observer)
- Vanilla design (no frameworks)

---

## 🎬 Interactions & Effects

-  Typing animation in hero section
-  Bounce-in section headings on scroll
-  Fade-in project cards on scroll
-  Hover effects on buttons and cards
-  Floating profile image animation
